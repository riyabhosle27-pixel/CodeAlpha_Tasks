Netflix Data Analysis (EDA)

This project performs Exploratory Data Analysis of the Netflix dataset.

Tools Used 
-Pandas
-Python
-Matplotlib
-Seaborn
Source of solving-Google Colab

Dataset Used
-Netflix shows/series AND Movies Dataset

Analysis Performed
-Top countries producing Netflix content
-Rating distribution
-Content released by year
-Movies vs TV Shows comparison

Insights
-Most Netflix content was released after 2015.
-Netflix has more movies than TV shows.
-The United States produces the most content.
-TV-MA is the most common rating.
